SignalBrief Privacy Policy

Effective date: 2026-02-25

Data We Collect

SignalBrief does not collect personal data from end users. The app accesses Facebook Pages and Instagram Business accounts that you explicitly authorize via Meta permissions.

How We Use Data

We use authorized access tokens only to publish SignalBrief posts and media to the connected Facebook Pages (and, when enabled, to Instagram).

Storage and Security

Access tokens are stored on the server with restricted file permissions and are not shared with third parties.

Sharing

We do not sell or share user data with third parties.

User Rights

You can revoke permissions at any time from your Meta account settings. Revoking access disables posting.

Contact

If you have any questions about this policy, contact: firstsignalbrief@gmail.com
